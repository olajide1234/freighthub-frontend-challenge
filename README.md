### Technologies used

 - ReactJS
   
  - Typescript
   
  - MobX
   
  - Webpack

  

### How to run

Steps to install

Clone this repository

    git clone https://github.com/olajide1234/freighthub-frontend-challenge.git

Install dependencies

    yarn install

Start the application in development mode

    yarn start:dev

Enjoy!!
